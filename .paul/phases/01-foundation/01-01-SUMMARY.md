## Accomplishments

- Gaussian copula (12×12 correlation matrix → Cholesky → norm.cdf) generates Likert values with realistic inter-trait correlations (e.g., energised/anxious r ≈ -0.50)
- Edge case injection is fully deterministic — friend pairs, anxiety outliers, low-profile users, near-identical twins, polar opposites all seeded deliberately from a shuffled index pool
- Three canned fixture files committed to data/synthetic/ (18, 23, 47 attendees) covering divides-evenly, remainder, and realistic-scale scenarios
- 200+ diverse names across 15+ cultural backgrounds in name_pools.py (no Faker dependency needed for names)

## Decisions Made

| Decision | Rationale | Impact |
|----------|-----------|--------|
| Model = RawQuizResponse (not Attendee) | SKILL spec names it this way; anonymiser.strip_pii() receives RawQuizResponse | All downstream code uses this type |

| name_pools.py instead of Faker for names | 200+ culturally diverse names without Faker overhead; reproducible | Faker dep still kept for future use |
