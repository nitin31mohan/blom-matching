<objective>
## Goal
Implement the proxy metrics layer: compute per-group pairwise similarity distributions and event-level aggregate metrics (mean cohesion, flag rate) from a GroupAssignment + AffinityMatrix.

## Purpose
Gives the Blom founder a quantitative picture of each matching run — beyond just cohesion scores and fit colours, the distribution stats expose whether a group's high mean is hiding a low-outlier pair. Event-level flag rate tracks how often the LLM review triggers across events, enabling regression detection if algorithm changes degrade quality.

    Implement `src/evaluation/metrics.py` with:

    **Schemas (Pydantic v2):**
    ```python
    class GroupSimilarityStats(BaseModel):
        group_id: str
        n_members: int
        mean_pairwise_sim: float
        std_pairwise_sim: float
        min_pairwise_sim: float
        max_pairwise_sim: float

    class EventMetrics(BaseModel):
        event_id: str
        n_groups: int
        mean_cohesion: float
        flag_rate: float        # fraction of groups with any flags
        groups: list[GroupSimilarityStats]
        computed_at: str        # ISO 8601
    ```

    **`compute_group_similarity_stats(group: Group, affinity: AffinityMatrix) -> GroupSimilarityStats`:**
    - If `len(group.user_ids) < 2`: return `GroupSimilarityStats` with all float fields = 0.0
    - Extract submatrix: `indices = [affinity.user_ids.index(uid) for uid in group.user_ids]`
    - Extract upper triangle (excluding diagonal): `np.triu(sub, k=1)` then mask/flatten to get all off-diagonal pairs
    - pairs = `sub[np.triu_indices(n, k=1)]` where n = len(indices)
    - Return mean, std, min, max of `pairs` (use `float(np.mean(...))` etc.)

    **`compute_event_metrics(assignment: GroupAssignment, affinity: AffinityMatrix) -> EventMetrics`:**
    - Call `compute_group_similarity_stats(g, affinity)` for each group
    - `mean_cohesion = float(np.mean([g.cohesion_score for g in assignment.groups]))`
    - `flag_rate = sum(1 for g in assignment.groups if g.flags) / len(assignment.groups)`
    - `computed_at = datetime.now(timezone.utc).isoformat().replace("+00:00", "Z")`

    Imports needed:
    - `from src.matching.assignment import Group, GroupAssignment`
    - `from src.matching.similarity import AffinityMatrix`
    - `import numpy as np`

    Avoid:
    - Re-implementing cohesion (use the affinity submatrix directly — same math as group_cohesion_score)
    - Raising on empty groups (guard with < 2 check)
  </action>
