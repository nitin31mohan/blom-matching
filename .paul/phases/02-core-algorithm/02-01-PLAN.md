<acceptance_criteria>

## AC-1: Matrix symmetry
```gherkin
Given an affinity matrix built from N feature vectors
When any entry matrix[i][j] is compared to matrix[j][i]
Then they are equal to within 1e-6
```

## AC-2: Diagonal is 1.0
```gherkin
Given an affinity matrix built from N feature vectors
When the diagonal is inspected
Then all diagonal entries equal 1.0 to within 1e-6
```

## AC-3: Score range
```gherkin
Given an affinity matrix built from any valid feature vectors
When all entries are inspected
Then all values are in the range [-1.0, 1.0]
```

## AC-4: Cohesion ordering
```gherkin
Given three groups:
  group_identical — all members have identical feature vectors
  group_mixed     — members drawn from varied profiles
  group_opposing  — members drawn from polar-opposite profiles
When group_cohesion_score is computed for each
Then cohesion(group_identical) > cohesion(group_mixed) > cohesion(group_opposing)
```

## AC-5: Marginal cohesion accuracy
```gherkin
Given a group of 4 users and a candidate user
When marginal_cohesion(affinity, candidate_id, group_user_ids) is called
Then the result equals the mean of the 4 individual similarity scores
  between the candidate and each group member
```

## AC-6: Fit colour thresholds
```gherkin
Given cohesion scores: 0.75, 0.55, 0.30
When group_fit_color() is called for each
Then 0.75 → "#22c55e" (great fit)
And  0.55 → "#f59e0b" (okay fit)
And  0.30 → "#ef4444" (poor fit)
```

    ```python
    class AffinityMatrix(BaseModel):
        model_config = ConfigDict(frozen=True, arbitrary_types_allowed=True)
        event_id: str
        user_ids: tuple[str, ...]   # pipeline_user_ids; index maps to matrix row/col
        matrix: np.ndarray          # shape (N, N), dtype float32
        computed_at: str            # ISO 8601
    ```
    Note: `frozen=True` prevents attribute reassignment; the numpy array itself
    is stored by reference (fine for read-only access in Phase 02).

    **Config loading:**
    ```python
    def _load_config() -> dict:
        """Load fit_thresholds from config/matching.yaml. Returns defaults if absent."""
    ```
    Load once at module import time into `_CONFIG`. Same pattern as encoder.py.

    **Functions to implement:**

    `build_affinity_matrix(feature_vectors: list[UserFeatureVector]) -> AffinityMatrix`
    - Extract user_ids list and stack vectors into shape (N, dim) numpy array (float32)
    - Vectors are already L2-normalised by encoder — cosine sim = dot product
    - Compute matrix = vectors @ vectors.T
    - Clip to [-1.0, 1.0] to guard against float32 drift
    - Set computed_at = datetime.utcnow().isoformat() + "Z"
    - Avoid: re-normalising vectors here (they're already normalised)

    `top_k_similar(affinity: AffinityMatrix, user_id: str, k: int = 5) -> list[tuple[str, float]]`
    - Find row index for user_id in affinity.user_ids
    - Extract that row, zero out the diagonal (self-similarity)
    - Return top-k (user_id, score) pairs sorted by score descending
    - Raise ValueError if user_id not in affinity.user_ids

    `group_cohesion_score(affinity: AffinityMatrix, group_user_ids: list[str]) -> float`
    - Find row/col indices for all group_user_ids
    - Extract the submatrix for those indices
    - Compute mean of all off-diagonal entries (i != j pairs)
    - Return 0.0 if group has fewer than 2 members
    - Avoid: including the diagonal (self-similarity = 1.0 would inflate scores)

    `group_fit_color(cohesion: float, config: dict | None = None) -> str`
    - Load thresholds from config or _CONFIG
    - great >= 0.68 → "#22c55e"
    - okay >= 0.42 → "#f59e0b"
    - poor < 0.42  → "#ef4444"

    `marginal_cohesion(affinity: AffinityMatrix, candidate_id: str, group_user_ids: list[str]) -> float`
    - Find row index for candidate_id
    - Find col indices for all group_user_ids
    - Return mean of affinity.matrix[candidate_row, group_cols]
    - Return 0.0 if group_user_ids is empty
    - Raise ValueError if candidate_id not in affinity.user_ids
