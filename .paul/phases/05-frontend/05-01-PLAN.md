    **`frontend/operator/src/canvas/GroupHull.tsx`:**
    - Uses `d3.polygonHull` to compute a convex hull from node positions.
    - Returns null if fewer than 3 nodes (hull undefined for 1–2 points — use circle/ellipse fallback).
    - Renders `<path>` with `fill` at 15% opacity, `stroke` at full color.

    ```typescript
    import { polygonHull } from 'd3-polygon'
    import type { SimNode } from '../types'

    interface Props {
      nodes: SimNode[]
      color: string
    }

    export default function GroupHull({ nodes, color }: Props) {
      if (nodes.length === 0) return null

      // Expand hull by padding each point outward from centroid
      const cx = nodes.reduce((s, n) => s + n.x, 0) / nodes.length
      const cy = nodes.reduce((s, n) => s + n.y, 0) / nodes.length
      const PAD = 40

      const points: [number, number][] = nodes.map((n) => {
        const dx = n.x - cx
        const dy = n.y - cy
        const dist = Math.hypot(dx, dy) || 1
        return [n.x + (dx / dist) * PAD, n.y + (dy / dist) * PAD]
      })

      if (points.length < 3) {
        // Fallback: circle around single/pair nodes
        return (
          <circle
            cx={cx}
            cy={cy}
            r={PAD + 20}
            fill={color}
            fillOpacity={0.12}
            stroke={color}
            strokeWidth={1.5}
            strokeDasharray="4 3"
          />
        )
      }

      const hull = polygonHull(points)
      if (!hull) return null

      const d = `M${hull.map((p) => p.join(',')).join('L')}Z`
      return (
        <path
          d={d}
          fill={color}
          fillOpacity={0.12}
          stroke={color}
          strokeWidth={1.5}
          strokeDasharray="4 3"
        />
      )
    }
    ```

    **`frontend/operator/src/canvas/ForceCanvas.tsx`:**
    - Accepts `assignment: GroupAssignment`, `attendees: Attendee[]`, `width: number`, `height: number`.
    - Builds `SimNode[]` from attendees, looks up group_id → fit_color from assignment.groups.
    - Runs `d3.forceSimulation` with:
      - `forceManyBody().strength(-300)` — node repulsion
      - `forceCollide(30)` — prevents overlap
      - `forceCenter(width/2, height/2)` — centres cluster
      - `forceLink` on pairs within the same group (strength 0.5, distance 80) — keeps groups together
    - On tick: copy simulation node positions to React state.
    - Renders: GroupHull components (background, below nodes), then AttendeeNode components.
    - Drag: `useRef` for SVG element; attach d3 drag behaviour via `useEffect`.

    Key pattern for avoiding stale closure issues:
    ```typescript
    const nodesRef = useRef<SimNode[]>([])
    const [, forceUpdate] = useState(0)  // tick counter to trigger re-render

    // in useEffect tick handler:
    simulation.on('tick', () => {
      forceUpdate(t => t + 1)
    })
    ```

    Avoid:
    - Calling `simulation.stop()` in the same tick as `simulation.restart()` — causes jitter
    - Creating a new simulation on every render — create once in useEffect with [] deps
    - Mutating node positions directly in React render path
