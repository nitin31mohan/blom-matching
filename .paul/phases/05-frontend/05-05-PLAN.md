<objective>
## Goal
Add a freeze/approve/straggler workflow to the operator canvas:
(1) Admin approves the current grouping — board freezes, Supabase notified (stub), nodes get green rings.
(2) Admin can thaw the board for manual adjustment at any time.
(3) Admin imports late sign-ups — each straggler is placed sequentially in best-fit group without
    disturbing frozen members — straggler nodes pulse yellow outward-wave.
(4) Groups that are stretched beyond their approved size show a capacity warning on the hull.

## Purpose
The algorithm sorts attendees ahead of the event. But sign-ups continue after the admin has
reviewed and locked in the groupings. Stragglers need a home without destabilising the groups
that everyone else has already been told they're in. The approval action also kicks off the
notification pipeline: Supabase compares the submitted assignment against the previous approved
state and notifies only attendees whose group has changed or who are being placed for the first time.

    **Create `src/lib/straggler.ts`:**
    Export one public function: `placeAllStragglers`.

    Algorithm: for each straggler (in order), try every group_id and compute `fitScore(straggler, groupId, testAttendees, profile, pairScores)` where `testAttendees` is the accumulating list (frozen + previously placed stragglers + straggler-under-test in that group). Pick the group_id with the highest score. Add the placed straggler to the accumulator before processing the next one.

    ```ts
    import { fitScore } from './fit'
    import type { Attendee, ActivityProfile, PairScoreMap } from '../types'

    export function placeAllStragglers(
      stragglers: Attendee[],
      frozenAttendees: Attendee[],
      groupIds: string[],
      activeProfile: ActivityProfile,
      pairScores: PairScoreMap,
    ): Attendee[] {
      // Returns stragglers with group_id filled in; frozenAttendees never mutated
      let current = [...frozenAttendees]
      const placed: Attendee[] = []
      for (const s of stragglers) {
        let bestGroupId = groupIds[0]
        let bestScore = -Infinity
        for (const groupId of groupIds) {
          const testNodes = [...current, { ...s, group_id: groupId }]
          const score = fitScore(
            { pipeline_user_id: s.pipeline_user_id, traits: s.traits },
            groupId,
            testNodes,
            activeProfile,
            pairScores,
          )
          if (score > bestScore) {
            bestScore = score
            bestGroupId = groupId
          }
        }
        const placed_s: Attendee = { ...s, group_id: bestGroupId, isStraggler: true }
        current = [...current, placed_s]
        placed.push(placed_s)
      }
      return placed
    }
    ```
