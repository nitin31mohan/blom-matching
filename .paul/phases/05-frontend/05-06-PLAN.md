## Goal
Group deletion via drag-to-bin interaction, dynamic GROUP_LAYOUT state, and full Pointer Events API migration for touch + mouse unification.

## Purpose
Admins need to dissolve a group and redistribute its members (treating them as stragglers). Touch/pointer events make the canvas usable on tablets — the primary target device for event-night admin use.

## Output

- `DEFAULT_GROUP_LAYOUT` in mock-data.ts; `GROUP_LAYOUT` stays as alias for zero-diff on ForceCanvas import

    **Hull handle drag tracking:**
    When `onDragHandlePointerDown` fires in GroupHull, ForceCanvas creates the handler:
    ```ts
    const makeHullDragHandler = useCallback((groupId: string) => (e: React.PointerEvent) => {
      e.preventDefault()
      e.stopPropagation()
      const svgEl = svgRef.current!
      svgEl.setPointerCapture(e.pointerId)
      setDraggingGroupId(groupId)

      const getPos = (ev: PointerEvent) => {
        const r = svgEl.getBoundingClientRect()
        return { x: ev.clientX - r.left, y: ev.clientY - r.top }
      }

      const onMove = (ev: PointerEvent) => setHullDragPos(getPos(ev))
      const onUp = (ev: PointerEvent) => {
        svgEl.removeEventListener('pointermove', onMove)
        svgEl.removeEventListener('pointerup', onUp)
        const pos = getPos(ev)
        // Dustbin zone: bottom-right 60×60px area
        if (pos.x > width - 70 && pos.y > height - 70) {
          onDeleteGroup?.(groupId)
        }
        setDraggingGroupId(null)
        setHullDragPos(null)
      }
      svgEl.addEventListener('pointermove', onMove)
      svgEl.addEventListener('pointerup', onUp)
    }, [width, height, onDeleteGroup])
    ```

    Pass to GroupHull: `onDragHandlePointerDown={makeHullDragHandler(gl.group_id)}`.
